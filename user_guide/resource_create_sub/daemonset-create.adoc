= Creating Daemon Sets
:toc:
:toc-title:

The following describes how to create daemon set resources.

== YAML Editor

Create a new daemon set by using the YAML editor.

. Click *[Create Daemon Sets]* in the <<../console_menu_sub/work-load#img-daemonset-main,Daemon Sets>> main screen.
. Enter configuration data about a daemon set to create in the YAML editor opened.
+
.Sample
[source,yaml]
----
apiVersion: apps/v1
kind: DaemonSet
metadata: 
  name: example <1>
  namespace: default <2>
spec: 
  selector: 
    matchLabels: <3>
      app: hello-hypercloud
  template: <4>
    metadata:
      labels: <5>
        app: hello-hypercloud
    spec:
      containers: <6>
        - name: hello-hypercloud <7>
          image: hypercloud/hello-hypercloud <8>
          ports: <9>
            - containerPort: 8080 <10>
----
+
<1> Daemon set name
<2> Name of a namespace where the daemon set is created
<3> Label of a pod to be managed by the daemon set
<4> Information about a pod to create
<5> Information about labels to use when classifying pods (Key-value format)
<6> Information about containers included in a pod
<7> Container name
<8> Container image
<9> Ports to expose to the container
<10> Port number to expose on the pod's IP address

. To create the resource with the data you entered, click *[Create]*.
